# **ENCODE()**

## **函数说明**

`ENCODE()` 函数用于对字符串进行对称加密。它通过结合一个秘钥来对字符串进行编码，并且解码时需要使用相同的密钥。需配合 [`DECODE()`](./decode.md) 解密。

## **函数语法**

```
> ENCODE (str, pass_str);
```

## **参数释义**

|  参数             | 说明                           |
|  --------------- | ------------------------------ |
| str              | 要编码的原始字符串。           |
| pass_str         | 用于加密的密码字符串（密钥）。    |

## **示例**

```SQL
mysql> SELECT ENCODE('hello', 'mysecretkey');
+----------------------------+
| ENCODE(hello, mysecretkey) |
+----------------------------+
| ?;?                         |
+----------------------------+
1 row in set (0.00 sec)
```
